Morsed

This is yet another Hardware PCB he made, and this one is a bit useful unlike Buzz. This is a simple card or a PCB card, through which we can decode and write Morse Code. In short: This is a Morse-Code Learning PCB!

The card's 3D front view

Features:

  • Live Decode: Here we press buttons, and it decodes it into letters only. (dash[-] and dot[.])
  • Word Decode: This is similar to the one above, and makes much more sense for using. The differernce is just that this makes up a whole word, and not just live decoding into letters.
  • A readable cheat-sheet where we can access every letter and it's corresponding Morse code.
  • Lastly we have a secret screen where if someone inputs 'GAME' in the Word Decode, it pushes to a simple game!
